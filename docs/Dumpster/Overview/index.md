# Dumpster
A micro-service to store and synchronize data between different components of the SDL ecosystem.  For example, an Android application's package name could be updated on SmartDeviceLink.com and synchronized between several OEM SDL Servers.

## Github
Dumpster is open sourced and managed on [Github](https://github.com/smartdevicelink/sdl_shaid_dumpster).  Please create an [issue](https://github.com/smartdevicelink/sdl_shaid_dumpster/issues/new) for any problems you encounter while using the service.
